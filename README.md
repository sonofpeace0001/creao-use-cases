## Deploy to GitHub + Vercel

### 1. Create the repo
Go to [github.com/new](https://github.com/new), name it `creao-use-cases`, and create it (do not add a README).

### 2. Push the files
Add `index.html`, `creao-wordmark.png`, `vercel.json`, `.gitignore`, and this `README.md` to the repo.

### 3. Import to Vercel
Go to [vercel.com/new](https://vercel.com/new), import your GitHub repo. Vercel auto-detects it as a static site — **leave build settings empty** and click **Deploy**.

### 4. Live URL
Your site will be live at `https://creao-use-cases.vercel.app` (or your custom domain).
